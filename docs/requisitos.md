# Requisitos funcionais
O sistema deve ser capaz de gerenciar clientes, produtos e vendas.  
Ele deve permitir que o usuário cadastre, atualize, exclua e consulte clientes e produtos.  
Além disso, deve permitir que o usuário realize vendas, o que inclui todo o processo de achar o produto, adiciona-lo no carrinho e realizar a compra, e gerar relatórios de vendas e clientes, que devem ser adicionados em arquivos **.pdf**.  

# Requisitos não funcionais
Tudo mencionado anteriormente deve ser gerenciado através de uma interface bonita e amigável para o usuário, permitindo que ele realize suas necessidades sem dor de cabeça.
