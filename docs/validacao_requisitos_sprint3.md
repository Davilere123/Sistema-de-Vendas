# Requisitos funcionais implementados
Todos os requisitos funcionais foram devidamente implementados e funcionam corretamente, sendo necessário apenas alguns polimentos.  
A única exceção é que o sistema gera recibos em PDF e relatórios de vendas, mas não **exatamente** de clientes.

# Requisitos não funcionais implementados
A interface está amigável e funcional. O usuários não deve ter problemas em fazer o que precisa. Porém, ainda falta muito polimento, então não está exatamente **bonita**.
